# Hyprland Configuration (v0.55+).
Hyprland is a dynamic tiling Wayland compositor developed by vaxerski and contributors.

> [!WARNING]
> **Important Syntax Notice:** Hyprland's legacy configuration syntax (`Hyprlang`) is officially deprecated. This setup has been fully migrated to the new **Lua-based** configuration system.

---

### Overview
This configuration stays close to the default Hyprland setup, focusing heavily on personal workflow efficiency and aesthetics. 

I am still learning Hyprland and its ecosystem. This configuration is not something I know perfectly from memory; it is the result of experimenting, modifying, and learning through the process.
A lot of the ideas and configurations here are inspired by other repositories and community projects. Whenever possible, I will credit the original creators.
